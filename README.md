## How to use

````dart
import 'package:flippo_navigation/flippo_navigation.dart';
````

Create an animated navigation drawer with the following snippet:

````dart
new Flippo(
      mask: new Scaffold(
        backgroundColor: Colors.white,
      ),
      body: new Scaffold(
        backgroundColor: Colors.white,
      ),
      drawer: new Scaffold(
        backgroundColor: Colors.black,
      ),
    );
````
